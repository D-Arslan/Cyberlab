Bienvenue dans le laboratoire d'injection SQL. Tu vas apprendre à exploiter l'une des vulnérabilités web les plus courantes et les plus dangereuses.

Une application web vulnérable (CyberShop) a été créée spécialement pour cet exercice. Ta mission : extraire des données sensibles en manipulant les requêtes SQL via les champs de saisie.

Les injections comme ' OR 1=1 -- ou ' UNION SELECT n'auront plus de secret pour toi.
